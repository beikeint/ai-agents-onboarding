## 🚀 3 步开始用

### 第 1 步:先装好环境(全免费,10-20 分钟)

| 软件 | 地址 | 干什么用 |
|---|---|---|
| **VSCode** | <https://code.visualstudio.com> | 您跟 AI 同事对话的工作台 |
| **Node.js 20+** | <https://nodejs.org>(下 LTS 版) | AI 工具箱的运行环境 |
| **Claude Code 插件** | VSCode 扩展市场搜 `Claude Code` | AI 同事的"大脑" |
| **Claude 订阅** | <https://claude.ai>(Pro $20/月) | AI 同事的"算力" |

### 第 2 步:打开 VSCode 的 AI 对话框,粘这段话

按 `⌃ + Esc`(Mac)或 `Ctrl + Esc`(Windows)打开 Claude Code 对话面板,粘以下内容回车:

```
帮我安装 AI 智能体系统,跑这条命令:

curl -fsSL https://raw.githubusercontent.com/beikeint/ai-agents-onboarding/main/onboarding.sh | bash
```

AI 会请求您点 **Allow**(允许),点一下,然后等 5-10 分钟,看到 `🎉 安装完成` 就好。

### 第 3 步:跟您的 AI 同事说"您好"

安装完成后,对话框粘:

```
打开 ~/ai-agents/web-ops,读 CLAUDE.md,然后告诉我你能做什么。
```

AI 运营经理会介绍自己能做的事。然后您就可以说中文下指令:

- 今天的网站巡检
- 这个月的运营报告
- 分析上周 Google 上哪些关键词掉了排名
- 帮我写一篇关于 [产品类目] 选型的英文博客
- 新产品上线,帮我做一个独立站(会自动切到 AI 建站总监)

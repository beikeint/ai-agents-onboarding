# 📱 给客户发的微信话术

public 模式下,客户**完全自助**。你发一条话术就行,不用中途介入。

---

## 🎯 默认模板(大部分客户直接发这个)

```
X总,给您装咱们的 AI 智能体系统,按顺序来,总共 30 分钟左右:

【第一步】装几个免费软件
1. VSCode:https://code.visualstudio.com
2. Node.js:https://nodejs.org(选 LTS 版)
3. VSCode 扩展市场搜 "Claude Code",装上
4. Claude 订阅:https://claude.ai,买 Pro 套餐(20 美金/月,智能体干活就靠它)

这 4 个装好跟我说一声 👍

【第二步】在 VSCode 打开终端,粘这行回车

打开方法:VSCode 顶部菜单 → 终端 → 新建终端
或按快捷键(Mac)Control + ~,(Windows)Ctrl + ~

curl -fsSL https://raw.githubusercontent.com/beikeint/ai-agents-onboarding/main/onboarding.sh | bash

全自动,5-10 分钟后看到 "🎉 安装完成" 就好。

【第三步】用起来
装完打开 VSCode,顶部菜单 → 终端 → 新建,粘这行回车:

code ~/ai-agents/web-ops

然后按 Control + Esc 打开 Claude Code,输入:
你好,请读 CLAUDE.md 介绍你自己

它会告诉您能做什么,然后说中文跟它聊就行。

任何一步卡住,截屏发我,我马上响应 💪
```

## 🙋 客户问"这个到底怎么用?"

```
简单说就是在 VSCode 里装一个 AI 助手,跟它聊天就能干活。

比如跟网站运营智能体说:
• "帮我做今天的网站巡检"
• "生成本月运营报告"
• "分析上周 Google 上哪些关键词掉了排名"

它会自动抓您的网站数据、分析、给建议,甚至直接帮您改站。
您只要做两件事:告诉它做什么 + 确认它做得对。

具体能力看每个智能体的 CLAUDE.md(5 分钟读完就懂)。
```

## ⚠️ 客户报错时你的回复模板

### 报错:"Node.js 太旧"

```
把 Node 升级到最新 LTS 版:
1. 去 https://nodejs.org 下载
2. 装好后重新跑那行 curl 命令
```

### 报错:"curl 超时"

```
切手机热点试一下(最稳),然后重新跑那行命令。
还不行的话截屏发我。
```

### 装完了但 Claude Code 看不到功能

```
MCP 还没配,我远程帮您弄。
约个时间,10 分钟搞定。
```

# 📱 给客户发的微信话术模板

复制粘贴,按客户情况填几个空即可。

---

## 模板 1:首次交付(完整版,适合大部分客户)

```
[客户称呼]您好,您的 AI 智能体系统已经开通,使用指南如下:

📦 您将得到什么
  · 独立站建站智能体(2-3 天建一个 B2B 外贸询盘站)
  · 网站运营智能体(每日巡检、月报、SEO 全自动)
  · 12 个 MCP 自动化工具
  · Astro 建站起点模板

📝 使用流程(约 15-30 分钟)

第 1 步:装环境(免费)
  ① VSCode(编辑器): https://code.visualstudio.com
  ② Node.js 20+: https://nodejs.org(选 LTS 版)
  ③ 在 VSCode 里装 Claude Code 插件(扩展市场搜 Claude Code)
  ④ 注册 Claude 订阅: https://claude.ai(Pro $20/月)

第 2 步:一键安装
  打开 VSCode → 打开终端(Mac 按 Ctrl+~)→ 粘这一行回车:
  
  curl -fsSL https://raw.githubusercontent.com/beikeint/ai-agents-onboarding/main/onboarding.sh | bash

第 3 步:脚本会让您生成一把"公钥",把它复制发给我
  我收到后 1-3 分钟内给您配好访问权限
  之后您按回车继续,脚本会自动装好所有东西

第 4 步:打开 VSCode,打招呼
  在 VSCode 里按 Ctrl+Esc 打开 Claude Code
  输入:"你好,请读 CLAUDE.md 介绍你自己"
  智能体会告诉您它能做什么

任何一步卡住就截屏发我,我马上响应 🚀
```

## 模板 3:客户问"这东西到底怎么用?"

```
简单说:
  · 打开 VSCode(一个免费编程软件)
  · 点开侧边栏的 Claude 图标
  · 像跟 ChatGPT 聊天一样,打字给它下指令

比如对网站运营智能体说:
  "帮我做今天的网站巡检"
  "生成本月运营报告"
  "分析上周哪些关键词排名掉了"

它会自动抓您网站的 Google 数据、写分析、给建议、甚至直接帮您改站。
您只要做两件事:①告诉它做什么 ②确认它做得对。

具体能力看每个智能体的 CLAUDE.md(5 分钟读完就懂)。
```

```
  "帮我选一个博客主题,要求:覆盖 [行业] 的 SEO 机会词"

💡 小贴士:
  · 说中文就行,智能体听得懂

  · 每天跟它说一句"每日巡检"就能自动跑完

遇到任何问题微信我,24 小时内响应 🚀
```
